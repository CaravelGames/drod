Ваша установленная версия ДРОДа может быть настроена/модифицирована путем создания соответствующих файлов или изменения файла Data/drod.ini следующими способами:


Графика

Вы можете продемонстрировать в игре новую или отличную графику путем редактирования строк в разделе [Graphics].

Выборочно откорректируйте спрайты внутриигровых ячеек, добавив:

General=<базовое название файлов моих ячеек>

Привнесите собственные стили оформления комнат, добавив новые записи, разделенные точкой с запятой, после следующей строки:

Style=<название моего стиля >

и добавив отдельную строку

<название моего стиля>=<базовое название файлов моего графического стиля>

Новые или существующие изображения неба могут быть отредактированы путем добавления строки:

<название моего стиля> Skies=<название файлов моего стиля неба для каждого уровня освещения>

Часы определяют базовое имя (имена) файлов изображений, предназначенных для отображения тревожных часов (счетчика оборотов). <basename>1 - файл изображения, предназначенный для отображения прироста в пол-оборота, а <basename>2 отображает прирост в целый оборот. Графика тревожных часов может быть заменена или дополнена изменением этой записи. Если указано несколько названий файлов, то, щелкнув во время игры по картинке часов, вы переключитесь на следующий набор их изображений.

Точно так же можно изменить изображения для существующих стилей.
Файлы изображений в подпапке Data/Bitmaps при желании также можно самостоятельно редактировать и заменять.

Поддерживаются изображения в форматах PNG, JPG, и BMP.


Звуковые эффекты

Воспроизводите различные звуковые эффекты, изменяя записи в разделе [Waves].
Вы можете либо заменять изначальные звуковые файлы своими собственными, либо добавлять в строку дополнительные файлы (через точку с запятой).

Поддерживаются звуковые форматы Ogg Vorbis и Wave.


Музыка

Воспроизводите в игре различные композиции, внося изменения в записи раздела [Songs].

Поддерживаются звуковые форматы Ogg Vorbis и Wave.


Другие возможности:

Раздел [Customizing]:
Установите AutoLogin=1, чтобы игра пропускала экран с логином игрока и автоматически выбирала последний профиль. Выбор игрока пропускается по умолчанию, когда в игре существует единственный профиль.
CrossfadeDuration определяет время смены картинки (максимум = 1000 миллисекунд).
Установите ExportSpeech=1, чтобы игра запрашивала ваше подтверждение перед переносом сценария реплик в отдельный HTML-файл при экспорте катакомбы. Используйте этот файл для удобства подготовки сценария реплик персонажа.
Установите FullScoreUpload=1, чтобы игра отсылала CaravelNet все результаты ваших побед, включая уже отправленные ранее, сразу после того, как вы щелкнете на кнопке "Отправить мои результаты" на экране настроек игрока. Рекорды утверждаются на сервере автоматически, и ваш онлайн результат будет обновлен сразу после их обработки. Совет: если вам не хватает результатов для комнат только из определенной катакомбы, щелкните на кнопке "Отправить результаты катакомбы" на экране смены локации, чтобы вновь отправить все рекорды демонстраций этой катакомбы.
Установите LogVars=1, чтобы выгрузить отчет о текущем состоянии переменных катакомбы во время игры в любой катакомбе, для которой ваш игрок редактировал разрешения.
Установите MaxDelayForUndo=<# миллисекунд >, чтобы изменить максимально допустимое время для отката внутриигровых движений. Подумайте об уменьшении значения этого параметра, если вы столкнулись со слишком большой задержкой при нажатии клавиши R для отката к контрольной точке или Backspace для отмены последнего хода. Уменьшив это число, откаты будут осуществляться значительно быстрее, однако сохранение в памяти более подробной информации о состоянии игры может потребовать больших ресурсов компьютера. В противном случае, увеличение значения данного параметра удлинит время ожидания отката в комнатах с большим количеством движений, но не потребует большего объема памяти для хранения дополнительной информации о состоянии игры. Слишком большое значение наглядно продемонстрирует вам, как ДРОД выглядел бы без движка 3.0.
Установите QuickPlayerExport=1, чтобы игра предоставила вам "быструю" возможность экспорта профиля игрока. Быстрый экспорт включает только сохраненные позиции уровня и начала комнаты, что обеспечивает небольшой размер файла и быструю скорость последующего импорта. Все основное продвижение в комнатах сохраняется, однако демонстрациями и сохраненными контрольными точками придется пожертвовать.
RoomTransitionSpeed - это время (в миллисекундах), которое уходит на кадрирование и подгонку параметров освещения при перемещении между комнатами. Изменение значения этого параметра варьирует длительность кадрирования комнаты и светового перехода. Установите значение равным 0, чтобы полностью отключить эти эффекты.
Установите ValidateSavesOnImport равным 0 для пропуска проверки корректности последовательностей игровых движений в сохраненных играх и демонстрациях при импорте файла игрока или обновлении катакомбы. Это может значительно ускорить процедуру импорта, однако сохраненные игры и демонстрации, записанные в предыдущих версиях импортированных катакомб, могут отображаться некорректно.
Windib: По умолчанию ДРОД использует драйверы Windib под Windows. Установите значение этого параметра равным 0, чтобы использовать вместо этого драйверы DirectX.

Раздел[Localization]:
Установите Keyboard=1, чтобы новые профили игроков использовали стандартные настройки клавиш движений для ноутбуков. При значении, равном 0, новые профили игроков будут использовать стандартные настройки настольных компьютеров.
Измените Language=Eng на другое трехбуквенное значение, чтобы игра перешла в режим другого языка, если тот поддерживается.
Установите ExportText=1, чтобы игра запрашивала ваше подтверждение перед тем, как экспортировать весь текст катакомбы для ее локализации. Добавляйте в этот файл дополнительные записи строчек, которым необходим перевод на иностранный язык, после строчек на родном языке; затем вновь импортируйте этот файл в вашу катакомбу, чтобы добавить поддержку другого языка.
Поддерживается текст в формате ASCII/UTF-8.

Локализация игрового движка.
Вы можете локализовать установленную версию ДРОДа, совершив следующие шаги:
1. Запустите ДРОД аргументом командной строки "-exportTexts <название файла>", чтобы сохранить все основные тексты игры (т.е., за исключением особых текстов катакомб) на текущем языке в определенный вами файл. Файл выгрузится в формате ASCII/UTF-8 с особенным синтаксисом для идентификации сообщений по номеру.
2. Переведите текст на другой язык, оставляя неизменным числовой код. Вы можете удалить любой текст или код, не подлежащий переводу.
3. Измените настройки языка в файле drod.ini на код переведенного языка.
4. Запустите ДРОД аргументом командной строки "-importTexts <название файла>", чтобы добавить или изменить указанный текст для текущих языковых настроек.
Для того чтобы помочь другим в локализации ДРОДа, поделитесь с ними своими файлами перевода и помогите осуществить шаги 3 и 4.

Использование консоли

Нажмите Enter во время игры или находясь в редакторе комнат, чтобы вызвать диалоговое окно чата/консоли. Вы можете использовать консоль для конфигурации настроек INI ДРОДа во время игры. Введите "~НазваниеРаздела:Настройка", чтобы увидеть текущее значение этой настройки. Введите "~НазваниеРаздела:Настройка=<значение>", чтобы изменить настройку. Определенное вами значение вступит в действие при следующем обращении к этой настройке. Введите "~help", чтобы вывести эту информацию в игре.


Медиа-файлы игры, созданные и находящиеся в собственности Caravel Games, не могут распространяться без письменного разрешения Caravel Games.

Более подробную информацию можно найти в Гостевой разработчиков на Форуме CaravelNet.
