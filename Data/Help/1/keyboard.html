Keyboard Commands

Keyboard commands available during the game are:

7 - move northwest 8 - move north 9 - move northeast
4 - move west 5 - wait 6 - move east
1 - move southwest 2 - move south 3 - move southeast
 
q - swing sword counter-clockwise (player turns left)
w - swing sword clockwise (player turns right)
 
Tab - Switch to next clone/squad member
. (keypad) - Special Command (scripted command key)
* (keypad) - Special Command 2 (scripted command key)
- (keypad) - Special Command 3 (scripted command key)
 
r - restart room (restart from checkpoint, if applicable)
Ctrl-r - restart room (restart from last different checkpoint)
Alt-r - restart room (skipping checkpoints)
Backspace - undo last move
+ (keypad) - Mirror last move ("battle" key)

These keyboard commands can be changed from the Player Settings Screen.

Movement macros: In addition holding down to the battle key to continuously repeat a two-step movement, pressing "Control" and a movement key together will perform a movement macro of that command and a move in the opposite direction immediately following. For example, press "Ctrl+Q" to rotate left and then right in quick succession. This is useful as a convenience to fight off large monster hordes. Additionally, press "Ctrl+5" to wait until the next monster spawn cycle when there are timed events on screen (i.e. the threat clock is showing).

Other in-game options are:

Esc - go back to Main Menu and pause the current game
Enter - display CaravelNet chat box and game stats: current hold, moves made, monsters killed, number of rooms conquered and explored, and totals for the current level.
Ctrl-Enter - display this DROD session's CaravelNet chat history.
Space - skip currently playing cutscene or dialogue
Shift - toggles room exit lock (enable to prevent accidentally exiting a room)
F1 - bring up this quick help screen
F2 - switch to the Player Settings Screen
F3 - toggle Puzzle Mode display
Alt+F3 - open Puzzle Mode options dialog
F4 - quick demo record
F5 - begin/end recording a demo
F6 - watch recorded demos
F7 - Toggle move counter display
Alt-F7 - Toggle framerate and sound memory usage display
Ctrl-F7 - Toggle script var change notifier (in holds you author, beta holds, or when playtesting)
F10 - toggle between full screen and windowed mode
F11 - save a screenshot to the player's export directory
Ctrl-F11 - save the room image only to the player's export directory

Clicking on door openers will highlight the yellow doors affected (green: opens, orange: toggles, red: closes).

Similarly, clicking on yellow doors will highlight what room elements affect them.

Right-clicking a room square will display its coordinates, names of game elements on this tile, monster move order, and more.

Double-clicking the minimap will display a larger map of all the explored rooms in the level. Press any key or click again to dismiss the level map.

Double-clicking the face portrait will bring up the history of the current room's in-game dialogue. Double-click a line or press OK to listen to any voice attached to that line again.

Click on the "?" button in the top-right corner of the game screen to open a browser window to get help for the current room.

Click on the adjacent numbered button to view a list of hold update notifications while connected to CaravelNet.
