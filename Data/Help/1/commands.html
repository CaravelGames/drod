Main Menu Commands

This is a summary of the commands shown on the main menu that appear when DROD first loads:

Play Game - Goes to the play sub-menu, which allows you to

  • Play Tutorial - Learn how to use the movement commands to play the game. Optionally learn about common game elements.
  • Play New Game - Start a new game from the beginning of the selected hold.
  • Continue Playing Game - Continues from the level and room you were at when you last exited DROD.  If you have never played the selected hold or have just completed the hold, this starts a new game from the first level.
  • Restore - Takes you to the Restore Screen, where you can restore a previously saved game.  Note that you can use the "Continue" command just to keep playing, but "Restore" is useful for returning to any room or level you have ever visited.
  • Change Location - Takes you to the Hold Selection Screen, where you can select which levels to play.
  • CaravelNet Chat - Takes you to the chat screen, where you can chat with other logged in CaravelNet users.
  • Back to Main Menu

Change Player - Takes you to the Player Selection Screen, where you can manage one or more player records.

Change Settings - Takes you to the Settings Screen, where you can fiddle with things.

Watch Demo - Watch a pre-recorded game demo.  Note that demos you record during play are accessible from the Demos Screen.

Build Levels - Takes you to the level editor, where you can design your own dungeons for anyone to play.

Get Help - If you are reading this, then you've mastered the help command.

Quit - You should spend all of your time playing DROD and should never have to exit the game so this command is irrelevant.
