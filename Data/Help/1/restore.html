Restoring Saved Games

The Restore Screen is reached by choosing "Restore" from the Main Menu.  You can restore to any one of several previous save points in the game.  Each save point stores information about the current room and play history, such as which rooms have been explored and conquered up to this point.  Saving is performed automatically as you explore the world of DROD, hence there is no save command.

The Restore Screen has the following options available:

Choose Level - Players can choose to restore to any of the levels they have visited.  Choose one from the level list to select it, or restart from the very beginning of the game by clicking the "Hold Start" button.

Choose Room - Click on the level map to select any room that's been visited previously.  Your journey is saved in each room at the point when the room was last entered before being conquered.  Rooms grayed out on the map are rooms that had not yet been visited when the currently selected room was last saved.  Click the "Level Start" button to start from the point when the level was first entered.

Choose Position - The mini-room shows the point from which the room will be restored.  Room checkpoints that were touched by the player are represented by "X" buttons.  Click on an "X" to restore from the last time this checkpoint was stepped on.  By default, the room will be restored at the latest checkpoint used.  Click the "Room Start" button to return to the time in your quest when the player last entered the room before conquering it.

Save point statistics - The properties for the currently selected save point are shown in the bottom-left corner of the screen.
Explored indicates how many rooms you have ever explored in the selected level.
Secrets found indicates the total secret rooms that have been conquered for this level across all saved games. This number can be used to check how many secret rooms you still need to complete before mastering the hold. You don't need to complete the secret rooms in the same savefile.
Total rooms conquered/explored indicates the total rooms conquered/explored in this hold for the currently selected save point. This is useful to make sure you have selected the latest save point while playing through a hold.

When a hold is completed, the Explored stat on the Restore Screen will change into a percentage of the rooms explored in that level, and Secrets completed will also indicate how many secret rooms exist for you to find in the selected level.

Click the "Restore" button to continue from the selected save point.  Note that restoring to a previous save point will return you back to that point in time in the level, with rooms that were explored or conquered after that point being reset to the way they were then.

Click the "Challenges" button to view a list of all scripted challenges you have completed in the current hold.

Click the "Export saves" button to export all your saved game data for the current hold. You can use this to share progress for that hold, without sharing author access to your levels.

Viewing demos - Press F6 to view demos recorded in the highlighted room.
