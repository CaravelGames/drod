CaravelNet Chat

This screen enables users logged in to CaravelNet to chat with one another.

Type in the text box at the bottom of the screen and hit Enter or click Send to post a message.

Chatting during play and editing: You may also enable chatting in-game by starting a game session, then hitting Enter to pop-up the game status dialog. Check the "Chat while playing" option. While playing, you may choose to receive all chatroom messages or only receive messages directed specifically to you by other players. You may likewise chat while in the room editor by pressing Enter. Press Ctrl-Enter to view this DROD session's chat history.

Console commands: Enter "~help" in the game or editor chat box to receive a list of client console commands for customizing your client at runtime.

Whispering: Click on one or more usernames in the user list to select only specific players to receive your messages. Unselect all highlighted names (Ctrl-click highlighted names) to broadcast messages to the entire chat room again.

Friends: You may mark people you enjoy chatting with as friends (see /friend macro below). When will be notified by the CaravelNet server when your friends come online.

Ignoring users: You may select one or more users in the chat room list and click "Ignore" to ignore them. When you flag a user as ignored, their name will be grayed out in the list and you will not see any more chats from them. You may "unignore" them at any time by selecting them and clicking Ignore again.

Chat command macros: Text beginning with a slash '/' is considered a command macro and is not sent as chat text. The word after the slash is a command to the chat interface, followed by optional parameters. Macro commands have short and long versions. Recognized commands include:

  • /f /friend (username) -- mark a user as a friend
  • /friendlist -- see your friend list
  • /help -- see the list of supported command macros
  • /i /ignore (username) -- cease receiving chats from a user
  • /ignorelist -- see your ignore list
  • /r /reply (text) -- whisper back to last user that whispered you
  • /u /unignore (username) -- stop ignoring a user
  • /uf /unfriend (username) -- unmark a user as a friend
  • /w /whisper (username) (text) -- send a private message to a user
