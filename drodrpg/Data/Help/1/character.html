Character Scripting

Non-player characters (NPCs) and scripts can be used to present a story and special events in your hold by defining a list of things to do and say. When an NPC is placed in a room, a dialog box will appear for customizing the NPC's appearance and behavior. On this dialog, a list box displays the character's script, which is initially empty. The script tells the NPC what to do when the player enters the room.

Script Commands. Read what script commands do and how to use them.

Graphic - Select what your NPC will look like from the list of available entities. Selecting "None" (the default) indicates this NPC is used for scripting events but cannot appear as a visible character in the room. Hint: With the "None" setting, a character cannot be made visible.

Character Management - Click "Characters" to bring up the Custom Character Management dialog.

Visible - If checked, the NPC will be visible when the player enters the room. Otherwise, it will be invisible. When an NPC is invisible, its script continues to play but it otherwise behaves as if it were not in the room. Hint: making a character with a "None" graphic visible will set it to the "Citizen 1" type.

Options - Click to set the NPC's options:

  • Ghost display, which makes a character appear on screen even when not set to be visible. Ghost display characters have no "physical" presence in the room, e.g. they can be walked through.
  • Minimap treasure, which makes a character count as a treasure for the purpose of automatically coloring the room on the minimap. This only applies if the character is visible or has ghost display.
  • _MyHue, which allows the predefined _MyHue var to be set for the character. Setting it this way allows it to be used when displaying an NPC's in-game appearance in the editor.
  • _MySaturation, which allows the predefined _MySaturation var to be set for the character. Setting it this way allows it to be used when displaying an NPC's in-game appearance in the editor.
  • _MyColor, which allows the predefined _MyColor var to be set for the character. Setting it this way allows it to be used when displaying an NPC's in-game appearance in the editor.
  • Speech Color, which controls the color of the character's speech subtitles. Each input takes a value of 0 to 255, and the three values are combined into a single RGB color value. If all inputs are zero, the default subtitle color will be used.
  • Processing sequence, which controls the order of scripts running in a room. Scripts with lower processing sequence values run earlier, and scripts with higher values run later. This does not affect the order in which equipment scripts will be run.

Add Command - Click to add another command to the script.

Delete Command - Deletes the selected commands from the script. Also performed by pressing the <Delete> key.

Reordering commands - During play, commands are performed in the order they are placed in the script. Hold down the mouse button on a command and drag it to reposition it in the script.

Cut-copy-paste commands - With one or more commands selected, press <Ctrl-X> to cut the commands from the script and place them in the script clipboard. Press <Ctrl-C> to make a copy of the selected script commands and place them in the script clipboard. Press <Ctrl-V> to paste the script commands in the script clipboard after the selected script command. The script clipboard is only valid while editing the current NPC.

Importing and exporting commands - Script commands may be imported from and exported to the operating system clipboard. Press <Ctrl-B> to export the currently highlighted script commands. Then you may paste these commands as text into a text editor. Press <Ctrl-Shift-B> to import a text selection from the OS clipboard. Command syntax must match that of exported commands in order for text to be imported correctly.

Click "Okay" to finish customizing the NPC.

You may click on the NPC in the room at any time to revise its appearance and script. Shift-click on an NPC to change its orientation. Click and drag an NPC to reposition it in the room.

Hold Alt to toggle viewing characters with the character icon, or as it will appear when playing. You can change the game setting Automatically Preview Characters to invert this toggle.
