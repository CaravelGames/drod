Restoring Saved Games

The Restore Screen is reached by choosing "Restore" from the Main Menu or pressing F3 or the "Load" button during play. You can restore to any save point recorded previously.

Display info: The Restore Screen displays a list of saved games for the active level set. The player's current stats and the current game state are displayed for the selected saved game. The number of total rooms your player has explored in the selected saved game is shown below the map, along with the number of hidden rooms discovered in this game.

You may manually create new save slots during play by pressing F2 or the "Save" button. Whenever you exit the game, a temporary "Continue" save slot is automatically created. Pressing "Continue Game" from the Main Menu will restore this saved game. Saving is also performed automatically at certain "Autosave" points during play. If you replay these parts of the game, only the latest version of each autosave is preserved.

The game also maintains a "Quicksave" slot for the level set (press F5 during play to quicksave). Press F9 to Quick Load this save slot.

The Restore Screen has the following options available:

Scorepoints - Opens up a list of all scorepoints in this level set. Scorepoints not yet achieved will be displayed as question marks if you did not author this level set. Achieved scorepoints will also show your best score.

Rename - Rename the selected save game.

Restore - Continue from the selected saved game. The "Continue" save slot will be overwritten with the current game session when you exit play.

Cancel - Return to the previous screen.

Export - Export the selected saved games to a file on disk.

Import - Import saved games from a file on disk.

Help - Display this help page.

Press Delete to permanently erase the selected saved games.
