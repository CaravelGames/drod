Game Commands

Keyboard commands available during the game are:

7 - move northwest 8 - move north 9 - move northeast
4 - move west 5 - wait 6 - move east
1 - move southwest 2 - move south 3 - move southeast
 
q - swing sword counter-clockwise (turn left)
w - swing sword clockwise (turn right)
 
r - restart room from the entrance
Backspace - undo moves you have made in the current room
+ (keypad) - View stats of all monsters in the room ("battle" key)
- (keypad) - View your score, and how it is calculated from your current statistics
0 (keypad) - Lock door at this tile (uses a key)
. (keypad) - Special command (when active)
Tab - Use equipment accessory (if carried)
Press Shift-Tab to activate special weapon and Ctrl-Tab to activate special armor.

These keyboard commands can be changed from the "Settings Screen".

Mouse commands: To move Tendry with the mouse, just click on the square you wish to move to. Tendry will walk there if there is a direct path. Click past the edge of the room to move into the next one. If you have a mouse wheel, it can be used to rotate Tendry. Inventory items may be activated, when applicable, by clicking on them in the side bar.

Other in-game options are:

Esc - go back to the Main Menu. Your game will pause and be saved to the "continue" slot.
Space - skip currently playing cutscene or dialogue
Enter - display CaravelNet chat box.
Ctrl-Enter - display this game session's CaravelNet chat history.
F1 - bring up this help screen
F2 - Save game
F3 - Load game
F5 - Quick save
F9 - Quick load
F4 - Edit game vars when playtesting
F6 - switch to the Player Settings Screen
F7 - Toggle move counter display
Alt-F7 - Toggle framerate and sound memory usage display
Ctrl-F7 - Toggle var change notifier (in holds you author, beta holds, or when playtesting)
F8 - Reload game sprites (for modders)
F10 - toggle between full screen and windowed mode
F11 - save a screenshot to the player's export directory
Ctrl-F11 - save the room image only to the player's export directory

Quick backtracking -- Press Ctrl+direction to run from wherever you are in the room to the closest exit in the indicated direction. This will only work if you have exited the current room in that direction before, and if nothing is blocking your path.

Right-click a room tile to display its coordinates and the names of game elements on this tile. It will also show contextual information. For instance, right-clicking door openers will highlight the doors affected (green: opens, orange: toggles, red: closes).

Right-clicking a monster will display its stats, any special properties, the damage you should expect to receive from fighting it, and the expected reward.

Clicking the minimap will display a larger map of all the explored rooms in the level. Clicking a room will temporarily display the current state of that room. Press any key or click off the map to return to play.

Mark rooms on the minimap by right clicking them. You may shade rooms with any of three colors to help you remember something important there.

Double-clicking the face portrait will bring up the history of the current room's in-game dialogue. Double-click a line to listen to any audio attached to that line again.
