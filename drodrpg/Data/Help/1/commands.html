Main Menu Commands

This is a summary of the commands shown on the main menu of the Title Screen:

Play Game - Goes to the play sub-menu, which allows you to

  • Play New Game - Start a new game from the beginning of the selected hold.
  • Continue Playing Game - Continues from the point you last played this hold.
  • Restore - Takes you to the Restore Screen, where you can restore a previously saved game.  Note that you can use the "Continue" command just to keep playing, but "Restore" is useful for returning to earlier save points.
  • Choose Level Set - Takes you to the Hold Management Screen, where you can select which hold (level set) to play.
  • CaravelNet Chat - Takes you to the chat screen, where you can chat with other logged in CaravelNet users.
  • Back to Main Menu

Change Player - Takes you to the Player Selection Screen, where you can manage one or more player profiles.

Change Settings - Takes you to the Settings Screen, where you can fiddle with things.

Build Levels - Takes you to the level editor, where you can design your own dungeons for anyone to play.

Get Help - If you are reading this, then you've mastered the help command.

Quit - You should spend all of your time playing DROD and should never have to exit the game so this command is irrelevant.
